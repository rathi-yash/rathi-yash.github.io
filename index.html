BitTorrent Client

A custom BitTorrent client implemented in Python, demonstrating peer-to-peer file sharing capabilities.

Python, Socket Programming, Multithreading

This project involves the development of a custom BitTorrent client from scratch, showcasing advanced networking and distributed systems concepts. The client implements core BitTorrent protocol features, enabling efficient peer-to-peer file sharing. Key features:

  • Torrent Parsing: Ability to parse .torrent files and extract essential metadata.
  • Peer Communication: Implements the BitTorrent peer wire protocol for data exchange.
  • Piece Management: Efficiently manages downloading and uploading of file pieces.
  • Multithreaded Design: Utilizes multithreading for concurrent connections and improved performance.
  • Tracker Communication: Interacts with BitTorrent trackers to discover and manage peers.
This project demonstrates proficiency in network programming and protocol implementation, showcasing skills in handling complex data structures and optimizing peer-to-peer communications.

Code

Soccer Analysis YOLO

A computer vision project using YOLO for real-time player detection and tracking in soccer matches.

Python, YOLO, OpenCV, Computer Vision

This project applies advanced computer vision techniques to analyze soccer matches in real-time. It utilizes the YOLO (You Only Look Once) object detection algorithm to identify and track players on the field. Key features:

  • Player Detection: Accurately identifies players in video frames using YOLO.
  • Real-time Tracking: Tracks player movements across consecutive frames.
  • Performance Analysis: Provides insights on player positioning and movement patterns.
  • Video Processing: Handles video input for continuous analysis of match footage.
  • Visualization: Generates visual output with bounding boxes around detected players.
This project showcases skills in computer vision, machine learning implementation, and video processing, demonstrating the ability to apply cutting-edge AI techniques to sports analytics.
